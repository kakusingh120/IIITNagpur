## IIITN Web

Steps to run on local :

Clone to local :
```bash
git clone https://github.com/Devendraxp/iiitnagpur.git
```
open folder
```bash
cd iiitnagpur
```
To install dependencies:

```bash
bun install
```

To run:

```bash
bun run app.js
```
or

```bash
nodemon app.js
```
For windows
```bash
npx nodemon app.js
```
